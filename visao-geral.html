Visão Geral do Projeto - FinControl

Sobre o Projeto

O FinControl é um sistema de Controle de Gastos Pessoal focado na facilidade de uso, permitindo a importação de faturas de cartão de crédito e organização financeira geral.

Stack Tecnológica

  • Backend: Java 21 + Spring Boot 3 + Maven
  • Banco de Dados: SQLite (via Spring Data JPA + Hibernate)
  • Frontend: React + Vite + TailwindCSS
  • Comunicação: REST API (JSON)

Como Executar

1. Executando o Backend

O backend utiliza Maven e Spring Boot. Para executá-plo, abra um terminal na pasta backend e execute:

cd backend
mvn spring-boot:run

A API estará disponível em: http://localhost:8080

Swagger (Documentação da API): http://localhost:8080/swagger-ui/index.html

2. Executando o Frontend

O frontend utiliza Vite. Para executá-lo, abra um terminal na pasta frontend e execute:

cd frontend
npm install
npm run dev

O painel estará disponível em: http://localhost:5173

Estrutura do Banco de Dados

O sistema utiliza as seguintes tabelas no SQLite (arquivo db/controle.db):

  • cartao: Cadastro de cartões de crédito.
  • fatura: Faturas importadas, com mês/ano de referência.
  • lancamento_cartao: Itens da fatura.
  • gasto_fixo: Financiamentos, assinaturas e contas de consumo.
  • investimento: Aportes mensais em investimentos.
  • categoria: Categorias para classificação dos gastos.
  • regra_categoria: Regras para auto-categorização de novos lançamentos baseadas em palavras-chave.
